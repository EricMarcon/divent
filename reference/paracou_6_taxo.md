# Taxonomy of Paracou plot 6 species

The taxonomy of species of the dataset
[paracou_6_abd](https://ericmarcon.github.io/divent/reference/paracou_6.md).
Distances in the tree are 1 (different species of the same genus), 2
(same family) or 3 (different families).

## Usage

``` r
paracou_6_taxo
```

## Format

An object of class
[ape::phylo](https://rdrr.io/pkg/ape/man/read.tree.html), which is a
phylogenetic tree.

## Source

Permanent data census of Paracou: <https://paracou.cirad.fr/>

## Details

This dataset is from Paracou field station, French Guiana, managed by
[Cirad](https://www.cirad.fr).
